🏠 LOCAL: Quando Usar

✅ USE quando

• Performance ultra-crítica
• Dados ultra-sensíveis
• App DEVE funcionar offline
• Cache de otimização
• Configurações do dispositivo
• Protótipos e MVPs

❌ NÃO use quando

• Colaboração necessária
• Sincronização multi-device
• Backup crítico obrigatório
• Analytics centralizados
• Dados compartilhados
• Escalabilidade infinita
// ✅ IDEAL para LOCAL UserDefaults.standard.set("dark", forKey: "theme") // ⚡ 2ms KeychainManager.save(password, for: "user") // 🔒 Ultra-seguro CoreDataManager.saveCacheLocally(images) // 📱 Offline-first
